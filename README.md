# VITarthi-Project-on-Rural-Health-Awareness
The following application is made to provide free consultation and awareness against common rural disease. It displays the causes symptoms and prevention methods of common rural diseases. It has a simple interface for non technical users. It is designed for educational purpose and can be expanded into a full rural health assistant system
Every year thousands of people in rural area suffer from diseases like diarrhea, maleria, skin infection, anemia, etc amd many die due to lack of awareness or availability of proper healthcare facilities.
This is a basic application that can help people in rural area get proper information about precautions, prevention, medication of some diseases that are common in rural area.
This is a miniature model of a mega project which can change healthcare forever. 
It will make healthcare affordable and accable for everyone.
